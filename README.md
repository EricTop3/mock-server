# mock-server
> node+koa模拟数据接口

*Koa需要v7.6.0以上node版本，低于此版本请先升级node*

### 使用到的库
+ `koa`
+ `koa-router`
+ `qs`

### 执行操作：
```
git clone https://github.com/daijingfeng/mock-server
cd mock-server
npm install
npm run mock
```
